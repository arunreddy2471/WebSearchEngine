ASP.Net is a web development platform provided by Microsoft. It is used for creating web-based applications. ASP.Net applications can also be written in a variety of .Net languages. These include C#, VB.Net, and J#. The ASP stands for Active Server Pages, and .NET is Network Enabled Technologies. What are the prerequisites for this .NET for beginners tutorial? You just require basic knowledge of .Net concepts. However, knowledge of programming language basics would be an additional help to learn this ASP.NET tutorials series.
